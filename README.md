# ECZANE VERİTABANI

Bu veritabanı tamamen çalışma amaçlı oluşturulmuştur. Kişi, TC kimlik numaraları, telefon numaraları, adresler ve e-mail adresleri tamamen uydurmadır.

Veritabanı aşağıdaki tablolardan oluşmaktadır.

-eczane
-ilac
-recete
-doktor
-personel
-adres
-uretici
-hasta
